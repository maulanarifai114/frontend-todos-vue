# Frontend Todos

## Prerequisite

### 1. Download Repository

```
git clone https://github.com/maulanarifai114/frontend-todos-vue.git
```

### 2. Install Package

```
npm install
```

### 3. Set up ENV

```
VUE_APP_API_URL = YOUR_API_URL
VUE_APP_ADMIN = YOUR_ADMIN_ID
```

### 4. Run Server

#### Development Mode

```
npm run serve
```

#### Production Mode

```
npm run build
```
